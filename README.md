# RAG & LLM AI Projects

This repository contains a collection of projects demonstrating the development of AI-powered applications using Large Language Models (LLMs), Retrieval-Augmented Generation (RAG), and AI agents.

These projects showcase practical implementations of modern AI engineering concepts including tool-calling agents, document retrieval systems, and intelligent automation.

## Projects Included

### 1. AI Job Hunter Agent

An AI-powered job search assistant that automatically finds and summarizes job opportunities using LLMs.

**Features**

* Search jobs from external sources
* Filter jobs based on skills and location
* AI-generated job summaries
* Multi-tool agent architecture

**Technologies**

* Python
* FastAPI
* Streamlit
* LLM APIs


### 2. RAG Document Question Answering System

A Retrieval-Augmented Generation (RAG) system that allows users to ask questions about documents using vector search and LLMs.

**Features**

* Document ingestion
* Vector embeddings
* Similarity search
* Context-aware answers using LLMs

**Technologies**

* Python
* LangChain
* Vector Databases
* LLM APIs

---

### 3. LLM Multi Tool AI Agent

A modular AI agent capable of selecting and executing tools dynamically to complete user tasks.

**Features**

* LLM-based tool selection
* Modular architecture
* Extensible tool framework
* AI reasoning workflows

**Technologies**

* Python
* FastAPI
* LLM APIs
* Agent architecture

## Learning Goals

* Build production-style AI applications
* Understand RAG architecture
* Implement multi-tool AI agents
* Integrate LLM APIs into backend systems
